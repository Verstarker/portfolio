About Travelmate

Travelmate was my first project using a mixture of Javascript, jQuery, and JSON. It was a travel app to help clients find a place to stay within New Zealand.

The user would input a city in New Zealand, add in the check-in and check-out days, and input how many guests they had with them. From this, the site would search through the JSON file and determine the places that fit within the fiter the user specified. Once the site had its list of viable places, it would display them.

This site used primarily a combination of jQuery and used a JSON file for all the information stored about each accomodation such as if it was a hotel or motel, the price of rooms, how many guests were allowed, and and additional services like a gym, parking, or breakfast.
